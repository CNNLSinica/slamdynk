ANN_Parity¶

class ANN_Parity.NeuralNetwork(images=None, parameters=None, validation=None, force_predict=False)¶

Keras/TensorFlow-based Neural Network model.

Uses TensorFlow as Keras’ backend to construct a neural network. This method also allows for GPU acceleration.

Parameters

  • images – List of ASE atoms objects that are being trained.

  • parameters – File of AMP or ANN-keras that contains parameters .

  • hiddenlayers – Structure of the neural network. Can either be in the format (int,int,int), where each element represnts the size of a layer and there and the length of the list is the number of layers, or dictionary format of the network structure for each element type. E.g. {‘Cu’: (10, 5, 5), ‘Pt’: (10, 5)}

  • activation – Activation type. (XXX Provide list of possibilities.)

  • maxTrainingEpochs (int) – Maximum number of times to loop through the training data before giving up.

  • batchsize (int) – Batch size for batch training.

  • parameters – Dictionary of parameters to be used in initialization. Mostly these are the same keywords as the keyword arguments in this function. This is primarily used to make saving/loading easier.

  • energy_coefficient (float) – Used to adjust the loss function; this is the initial weight applied to the energy component.

  • force_coefficient (float or None) – Used to adjust the loss function; this is the initial weight applied to the force component. Note you can turn off force training by setting this to None.

  • convergenceCriteria (dict) – Dictionary of convergence criteria, analagous to the main AMP convergence criteria dictionary.

  • ADAM_optimizer_params – Dictionary of parameters to pass to the ADAM optimizer. See https://www.tensorflow.org/versions/r0.11/api_docs/python/ train.html#AdamOptimizer for documentation
