Add vacancies into Atoms object

Parameters

  • atoms (ASE Atoms object) –

  • vac_count (dict) –

  • vac_prob (dict) – use either vac_count or vac_prob

  • rand (bool or int) – random seed

Returns

new_atoms

Return type

ASE Atoms object

Example

>>> from pkgall.structools import assign_alloy, add_vac
>>> from ase.build import bulk
>>> Cu=bulk('Cu',cubic=True)*(2,2,2)
>>> Alloy=assign_alloy(atoms=Cu, elements=['Cu','Al'])
>>> print(Alloy)
Atoms(symbols='Cu16Al16', pbc=True, cell=[7.22, 7.22, 7.22])
>>> Vac=add_vac(atoms=Alloy, vac_count=3, vac_prob={'Cu':0.5, 'Al':0.9})
>>> print(Vac)
Atoms(symbols='Cu7Al', pbc=True, cell=[7.22, 7.22, 7.22])
