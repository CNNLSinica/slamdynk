Welcome to SLAMDYNK3's documentation!
=====================================

The SLAMDYNK3 is a set of Python_ modules which combines serveral packages including 
ASE_, Amp_, LAMMPS_ and VASP_ to train a Self-Learning Artificial neural network potential, 
and on-the-fly using the potential to run Monte-Carlo simulations.
