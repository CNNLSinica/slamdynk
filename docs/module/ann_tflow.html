ann_tflow¶

class ann_tflow.NeuralNetwork(images=None, parameters=None, sampling=None, hiddenlayers=(5, 5), cutoff=6.5, maxTrainingEpochs=1000000, opt_num_correction_pairs=1000, opt_parallel_iterations=8, stdscaler='normalize', cores=1, multi_gpus=False, gpu_max_usage=4096, energy_coefficient=1.0, force_coefficient=0.04, convergenceCriteria={'energy_maxresid': None, 'energy_rmse': 0.001, 'force_maxresid': None, 'force_rmse': None})¶

TensorFlow-based Neural Network model.

Uses TensorFlow 2.0 to construct a neural network. This method also allows for GPU acceleration.

Parameters

  • images (str) – List of ASE atoms objects that are being trained.

  • parameters (str) – File of AMP or ANN-tflow that contains parameters.

  • sampling (str or int) – sampling method of data point in images

  • cutoff (float) – Cutoff represent the radius above which neighbor interactions are ignored. Default is 6.5 Angstroms.

  • hiddenlayers (tunple or dict) – Structure of the neural network. Can either be in the format (int,int,int), where each element represnts the size of a layer and there and the length of the list is the number of layers, or dictionary format of the network structure for each element type. E.g. {‘Cu’: (10, 5, 5), ‘Pt’: (10, 5)}. Default is (5, 5).

  • maxTrainingEpochs (int) – Maximum number of times to loop through the training data before giving up. Default is 1,000,000.

  • opt_num_correction_pairs (int) – Specifies the maximum number of (position_delta, gradient_delta) correction pairs to keep as implicit approximation of the Hessian matrix in tensorflow probability l-bfgs optimizer. Default is 1,000.

  • opt_parallel_iterations (int) – The number of iterations allowed tensorflow probability l-bfgs optimizer to run in parallel. Default is 8.

  • stdscaler (str) – Normalize method of input fingerprints/fingerprint-primes. 2 method are allowed, normalize and zero_mean. Default is ‘normalize’.

  • cores (int) – The number of cores for calculating fingerprints/fingerprint-primes by AMP. Default is 1.

  • multi_gpus (bool) – Set this to True allows using multiple GPUs in training procedure. Default is False.

  • gpu_max_usage (int) – Maximum memory (in MB) to allocate on the GPUs device. Default is 4096 MB.

  • energy_coefficient (float) – Used to adjust the loss function; this is the initial weight applied to the energy component. Default is 1.0.

  • force_coefficient (float or None) – Used to adjust the loss function; this is the initial weight applied to the force component. Default is 0.04.

convergenceCriteria: dict

Dictionary of convergence criteria, analogous to the main AMP convergence criteria dictionary. Default is {‘energy_rmse’: 0.001, ‘energy_maxresid’: None,

‘force_rmse’: None, ‘force_maxresid’: None}

Note: you can turn off force training by setting both ‘force_rmse’ and ‘force_maxresid’ to None.
